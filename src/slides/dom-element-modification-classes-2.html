Коллекция classList

classList - это еще один псевдомассив, на этот раз реализующий прототип DOMTokenList. Здесь у нас в наличии чуть больше методов чем мы видили в псевдомассивах ранее. С решением 95% задач можно справится не уходя за рамки данного прототипа. Среди основных методов можно выделить:

  • add(className) - добавляет (если не добавлен) класс с именем className
  • remove(className) - удаляет (если есть) класс с именем className
  • contains(className) - проверка на наличеие класса с именем className
  • toggle(className) - добавляет если нету и удалет если есть класс с именем className
