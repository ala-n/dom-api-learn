JavaScript

Ядро JavaScript расширяется для различных целей путем добавления в него новых объектов, определяющих API для взаимодействия с внешним миром, например:

  • JavaScript на стороне клиента расширяет ядро языка, предоставляя объекты для контроля браузера (Browser Object Model (BOM)), а также непосредственно Document Object Model (DOM), о которых мы поговорим далее.
  • JavaScript на стороне сервера расширяет ядро языка, предоставляя объекты обращения к файловой системе, обмена информацией по протоколу HTTP и т.п.
  • JavaScript в базах данных в свою очередь расширен объектами доступа к элементам этих баз данных.
