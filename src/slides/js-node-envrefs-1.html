Доступ к окружению элемнтов страницы

Помимо описанных ранее методов для получения произвольного элемента в дереве сам интерфейс (прототип) Node содержит динамические ссылки на все "окружение" узла дерева: родительский элемент (parentNode), коллекцию дочерних элементов (childNodes), ссылки на первый/последний дочерний элемент (firstChild/lastChild), ссылки на соседей в родительском элементе (previousSibling / nextSibling). Визуально можно просмотреть доступные связи узла на следующей схеме.
