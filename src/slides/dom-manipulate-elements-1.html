Изменеие иерархии элементов

  • appendChild(node) [Node] - Вставляет node как последний дочерний узел данного элемента.
  • insertBefore(newNode, referenceNode) [Node] - Вставляет newNode непосредственно перед referenceNode, потомком данного элемента.
  • replaceChild(newNode, oldNode) [Node] - Заменяет одного потомка (newNode) из существующего на второй (oldNode).
  • removeChild(node) [Node] - Удаляет дочерний элемент node из DOM. Возвращает удаленный элемент.
