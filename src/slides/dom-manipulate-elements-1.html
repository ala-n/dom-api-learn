Изменеие иерархии элементов

  • appendChild [Node] - Вставляет Node как последний дочерний узел данного элемента.
  • insertBefore [Node] - Вставляет первым Node данный в качестве параметра, непосредственно перед вторым, потомком данного элемента Node.
  • replaceChild [Node] - Заменяет одного потомка Node из существующего на второй указанный в параметре.
  • removeChild [Node] - Удаляет дочерний элемент из DOM. Возвращает удаленный элемент.
